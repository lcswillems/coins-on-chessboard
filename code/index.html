Pièces sur un échiquier

Formalisme

Une configuration s'écrit de la forme {prison1},...,{prisonK}|{pièce1},...,{pièceN} où, par exemple, {prison1} vaut A1 et {pièce1} vaut B1c, B1t ou B1h avec :

  • c pour désigner une pièce de face quelconque
  • h pour désigner une pièce de face "head"
  • t pour désigner une pièce de face "tail"

Une séquence de coups s'écrit de la forme {coup1},...,{coupM} où, par exemple, {coup1} vaut A3.

Un jeu est composé d'une configuration initiale et d'une séquence de coups et s'écrit de la forme {configuration}|{séquence}.

Comment jouer ?

2 modes sont actuellement présents :

  • "édition" pour modifier la configuration initiale;
  • "jeu" pour jouer sur la configuration initiale

Dans le mode "édition", lorsque vous vous placez sur une des cases de l'échiquier :

  • un clique gauche fait apparaître ou disparaître une prison;
  • appuyer sur "c", "t", "h" fait apparaître ou disparaître une pièce de type "c", "t" ou "h".

Dans le mode "jeu", lorsque vous vous placez sur une des cases de l'échiquier :

  • un clique gauche déplace la pièce selon les règles du jeu;
  • un clique droit annule le dernier coup.
